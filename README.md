Detailed Instructions

You are stuck in an abandoned trafficking facility. You must do everything you can to escape the facility before you are caught.

WHAT YOU NEED TO DO: Find keycards scattered across the map in order to open the doors leading to the exit. If you explore further, you can find gun collectibles also scattered around the map which can increase your total score. 

Platform: PC only. Hardware: Laptop with RTX 500 ADA and INTEL Core ULTRA 7. 32GB RAM and above

Bugs: Doors always play the closing animation first when the game starts
